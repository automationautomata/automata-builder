Отлично — построим явную функцию (f(\tilde x)) из теоремы 3 для твоего автономного автомата и поясним шаги.

**Данные автомата**

* Выходный алфавит (O={0,1}) → положим (как в предыдущих примерах) числовое отображение
  (\beta(0)=1,; \beta(1)=2). Тогда (m=2).
* Множество состояний (S={s_0,s_1},; k=2).
* Переходы: (s_0\mapsto s_1,; s_1\mapsto s_1).
* Выходы: в (s_0) даётся символ (1), в (s_1) — символ (0).
* Начальное состояние возьмём (s_0).

Отсюда последовательность выходных символов (в числовой кодировке (\beta)) по шагам:
[
i_1=\beta(1)=2,\quad i_2=\beta(0)=1,\quad i_3=1,;i_4=1,\dots
]
(первый шаг — выход 2, далее бесконечные единицы).

---

### Шаг 1. Связь (\tilde x_k) с номером шага (k)

В доказательстве теоремы для (\tilde x_k) (для рассматриваемой кодировки последовательностей состояний) получается
[
\tilde x_k = 1+\frac12+\frac1{2^2}+\dots+\frac1{2^{k-1}} = 2-\frac{2}{2^{,k}}.
]
Отсюда выражаем (k) через (\tilde x):
[
2^{,k}=\frac{2}{2-\tilde x}\quad\Rightarrow\quad
k=\log_2!\frac{2}{2-\tilde x}.
]
(домен: (1\le \tilde x<2), при (k\to\infty) получаем (\tilde x\to2^{-}).)

---

### Шаг 2. Формула для (\tilde y_k)

По формуле (аналогично тексту теоремы)
[
\tilde y_k = i_1 + \frac{i_2}{m+1} + \frac{i_3}{(m+1)^2} + \dots + \frac{i_k}{(m+1)^{,k-1}}.
]
Здесь (m=2\Rightarrow m+1=3). Подставляем нашу последовательность (i_1=2,; i_2=i_3=\dots=1). Тогда для (k\ge1):
[
\tilde y_k = 2 + \sum_{j=1}^{k-1}\frac{1}{3^{,j}}.
]
Сумма геометрической прогрессии даёт
[
\sum_{j=1}^{k-1}\frac{1}{3^{j}}=\frac{\frac{1}{3}\bigl(1-(\tfrac{1}{3})^{k-1}\bigr)}{1-\tfrac{1}{3}}
=\tfrac12\bigl(1-(\tfrac{1}{3})^{k-1}\bigr).
]
Итог:
[
\tilde y_k = 2 + \tfrac12\bigl(1-(\tfrac{1}{3})^{k-1}\bigr)
= \tfrac{5}{2} - \tfrac12\bigl(\tfrac{1}{3}\bigr)^{,k-1}.
]
Предел при (k\to\infty): (\tilde y_\infty = 5/2 = 2.5).

---

### Шаг 3. Подстановка (k(\tilde x)) → функция (f(\tilde x))

Подставляем (k=\log_2\frac{2}{2-\tilde x}) в выражение для (\tilde y_k). Получаем функцию
[
\boxed{,f(\tilde x)=\tilde y(\tilde x)
=\frac{5}{2}-\frac{1}{2}\Bigl(\frac{1}{3}\Bigr)^{,\log_2\frac{2}{2-\tilde x}-1},}
\qquad(1\le\tilde x<2).
]

Можно слегка упростить множители:
[
\Bigl(\frac{1}{3}\Bigr)^{\log_2\frac{2}{2-\tilde x}-1}
=3^{1}\cdot 3^{-\log_2\frac{2}{2-\tilde x}}
=3\cdot 3^{-\log_2\frac{2}{2-\tilde x}}.
]
Так что эквивалентная запись
[
f(\tilde x)=\frac{5}{2}-\frac{3}{2};3^{-\log_2!\frac{2}{2-\tilde x}}.
]
Или, используя степенную запись через основание (\frac{2}{2-\tilde x}):
[
f(\tilde x)=\frac{5}{2}-\frac{3}{2}\Bigl(\frac{2}{2-\tilde x}\Bigr)^{-\log_2 3}.
]

---

### Проверка на отдельных (k)

* (k=1): (\tilde x_1 = 2-2/2=1). Тогда (\tilde y_1 = 5/2 - \tfrac12(1/3)^{0}=2.0) (что соответствует (i_1=2)).
* (k=2): (\tilde x_2 = 2-2/4=1.5). Тогда (\tilde y_2 = 5/2 - \tfrac12(1/3)^{1}=2+\tfrac13=\tfrac{7}{3}\approx2.333).
* При (k\to\infty): (\tilde x\to2^{-}), (f(\tilde x)\to 2.5).
